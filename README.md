# 四大名著经典学习网

## 项目介绍

**四大名著经典学习网**是一个专注于中国古典四大名著（《三国演义》《水浒传》《红楼梦》《西游记》）的在线学习平台。网站以"寓教于乐"为核心理念，通过答题闯关、段位晋升、徽章收集、积分商城等游戏化机制，帮助用户在趣味互动中深入了解四大名著的故事情节、人物关系、文化内涵和历史背景。

## 功能说明

### 用户系统
- **注册/登录**：支持用户名+密码注册登录
- **游客模式**：无需注册即可体验平台核心功能，降低使用门槛
- **个人中心**：展示学习统计（答题数、正确率、学习时长、掌握知识点数等）

### 四大名著学习
每部名著设有独立的学习入口页面，包含：
- **三阶段递进学习**：从基础到进阶，循序渐进掌握名著内容
  - 第一阶段：入门基础
  - 第二阶段：进阶提升
  - 第三阶段：深度挑战
- **原著阅读**：提供四大名著的原文阅读功能，支持章节导航

### 答题练习
- **自由选题**：支持按名著、按阶段灵活选择答题范围
- **题量可配**：可根据需求选择不同题目数量
- **即时反馈**：每题作答后立即显示对错与解析
- **成绩统计**：答题结束后展示得分、正确率等详细数据

### 段位系统
- 根据答题表现和积分累积，用户可逐步晋升段位
- 段位进度条直观展示当前等级与下一等级的差距
- 段位图标采用古典风格设计，与整体视觉统一

### 徽章成就
- 丰富的徽章体系，涵盖：
  - **答题成就**：100题 / 500题 / 1000题 / 2000题徽章
  - **名著专属**：三国读者、水浒读者、红楼读者、西游读者
  - **全通成就**：通读四部名著的全书读者徽章
  - **特色徽章**：水浒一百零八将、三国谋士、红楼人物、西游情节、诗词达人等
- 徽章墙集中展示已获得和未获得的徽章

### 排行榜
- 多维度排行榜：总排行、各名著专属排行
- 展示用户排名、积分、答题数等数据
- 激励用户之间的良性竞争

### 积分商城
- 通过答题和学习获取积分
- 积分可用于商城兑换虚拟道具或特权
- 积分余额与获取记录清晰展示

### 争议话题
- 汇集四大名著中的经典争议话题供用户探讨
---

## 项目亮点

### 古风美学设计
整体 UI 采用中国传统美学风格：宣纸底色、棕金色调、朱砂红点缀，搭配竹青绿与靛蓝，营造沉浸式的古典文化氛围。页面装饰有落叶飘落动画、金色渐变分割线、水墨纹理背景等细节，让学习过程充满仪式感。

### 游戏化学习体验
将枯燥的文学知识学习转化为闯关游戏：段位晋升带来成长感，徽章收集激发探索欲，积分商城提供即时奖励，排行榜营造竞技氛围。多层次的激励机制让用户保持持续学习的动力。

### 三阶段递进体系
每部名著都设计了三个难度递进的学习阶段，从入门到精通，适配不同知识水平的用户。初学者可以从基础阶段建立信心，资深读者可以挑战高阶内容检验功底。

### 完整的名著生态
不仅覆盖全部四大名著，还提供原著阅读、争议话题讨论等延伸功能，构建了一个从"学"到"读"再到"思"的完整学习闭环。
