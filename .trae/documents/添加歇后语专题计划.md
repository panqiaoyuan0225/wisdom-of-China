# 在xiyou\_stage2.html中添加歇后语专题计划

## 任务概述

在`xiyou_stage2.html`中新建一个"拾贰 · 歇后语趣谈"专题，包含西游记相关的歇后语题目，仿照`sanguo_stage1.html`的题目格式。

## 需要修改的内容

### 1. 在侧边栏添加新导航项

* 文件：`d:\桌面\MyWeb\xiyou_stage2.html`

* 位置：`.sidebar` div内

* 操作：在"拾壹·火焰山受阻"后添加新的导航项：

```html
<div class="nav-item" onclick="changeTab(this, '拾贰 · 歇后语趣谈')">拾贰·歇后语趣谈</div>
```

### 2. 在contentData中添加新专题内容

* 文件：`d:\桌面\MyWeb\xiyou_stage2.html`

* 位置：`contentData`对象内

* 操作：添加`'拾贰 · 歇后语趣谈'`的内容

### 3. 题目格式设计

参考`sanguo_stage1.html`的题目格式，设计歇后语题目：

* 封面页：展示专题标题和开始按钮

* 题目页：使用选择题形式展示歇后语

* 每道题包含：题目、选项、答案

### 4. 歇后语题目内容

将用户提供的歇后语整理成选择题形式，分为四个系列：

* 孙悟空系列（10题）

* 猪八戒系列（10题）

* 唐僧系列（6题）

* 其他人物与情节（14题）

## 实施步骤

1. 在侧边栏添加新导航项
2. 在contentData中添加歇后语专题的封面和题目内容
3. 添加JavaScript代码处理题目显示和答案查看功能
4. 测试验证功能正常

## 预期结果

* 用户可以在侧边栏点击"拾贰·歇后语趣谈"进入专题

* 专题包含封面页和题目页

* 用户可以查看答案和解析

* 整体风格与页面其他部分一致
