# 答题设置页面开发计划

## 目标
在 `quiz.html` 中创建一个答题设置页面，作为答题系统的入口界面。

## 设计要求
- **背景**：米黄色 `#f5f5dc`（与 index.html 一致）
- **功能**：设置页面，按钮可点击但暂不开始测试
- **风格**：简洁美观，不仿照 honglou_stage3

## 页面布局

```
┌─────────────────────────────────────────┐
│                                         │
│           📜 答题练习                    │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │                                 │   │
│  │   选择模式                       │   │
│  │   [📚 复习模式]  [🌟 拓展模式]   │   │
│  │                                 │   │
│  │   题目数量                       │   │
│  │   [5] [10] [15] [20]            │   │
│  │                                 │   │
│  │   是否限时                       │   │
│  │   [开启] [关闭]                 │   │
│  │                                 │   │
│  │      [开始答题]                 │   │
│  │                                 │   │
│  └─────────────────────────────────┘   │
│                                         │
└─────────────────────────────────────────┘
```

## 实现步骤

### 步骤 1：页面基础结构
- 设置米黄色背景
- 添加标题"答题练习"
- 创建返回主页链接

### 步骤 2：设置卡片容器
- 居中显示的白色卡片
- 圆角边框、阴影效果

### 步骤 3：模式选择区域
- 两个按钮：复习模式、拓展模式
- 点击可选中（高亮状态）

### 步骤 4：题目数量选择
- 四个选项：5、10、15、20
- 点击可选中

### 步骤 5：限时开关
- 开启/关闭两个按钮
- 点击切换状态

### 步骤 6：开始答题按钮
- 大按钮，点击后弹出提示"题库开发中，敬请期待"
- 悬停动效

## 配色方案

| 元素 | 颜色 |
|------|------|
| 页面背景 | `#f5f5dc` |
| 卡片背景 | `#ffffff` |
| 标题颜色 | `#8b4513` |
| 按钮边框 | `#333` |
| 选中状态 | `#d4af37`（金色） |
| 按钮文字 | `#8b4513` |

## 交互动效
- 按钮悬停：上浮 + 阴影加深
- 选中状态：金色边框 + 背景变化
- 开始按钮点击：弹窗提示
