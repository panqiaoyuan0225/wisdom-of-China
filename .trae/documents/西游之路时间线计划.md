# 计划：为 xiyou\_stage3.html 添加"西游之路"时间线内容

## 任务目标

将用户提供的西游记时间线内容，仿照 shuihu\_stage3.html 中"拾捌 · 英雄之路"的时间线格式，添加到 xiyou\_stage3.html 的"拾捌 · 西游之路"部分。

## 分析

### shuihu\_stage3.html "英雄之路"格式特点：

1. 使用 CSS 时间线样式（timeline）
2. 左侧有垂直时间线（金色渐变）
3. 每个事件包含：

   * **年份/回数**（左侧，右对齐）

   * **圆点标记**（在时间线上）

   * **事件标题**（加粗）

   * **事件描述**（`<b>事件：</b>`开头）

### 用户提供的西游记时间线内容（共25个事件）：

从约公元前22世纪（大禹治水）到约公元661年（唐僧最后一次回长安）

## 实施步骤

### 步骤1：添加时间线 CSS 样式

在 contentData 的"拾捌 · 西游之路"内容中添加 `<style>` 标签，包含以下样式：

* `.timeline-container` - 时间线容器

* `.timeline-line` - 垂直时间线（使用西游记主题色：青色/金色）

* `.timeline-event` - 事件容器

* `.event-year` - 年份样式

* `.event-dot` - 圆点标记

* `.event-content` - 内容区域

* `.event-title` - 标题样式

* `.event-desc` - 描述样式

### 步骤2：创建时间线 HTML 结构

将25个时间线事件按时间顺序排列，每个事件包含：

1. 年份（如"约公元前22世纪"）
2. 事件标题
3. 事件描述

### 步骤3：调整颜色主题

将时间线颜色调整为西游记主题色：

* 时间线：从金色(#FFD700)到青色(#008b8b)渐变

* 圆点：金色(#FFD700)

* 标题：深红色(#C02C3F)或青色(#008b8b)

* 年份：青色(#008b8b)

## 预期结果

用户点击"拾捌 · 西游之路"时，将看到一个完整的时间线，展示从大禹治水到唐僧取经成功的完整历史脉络。
