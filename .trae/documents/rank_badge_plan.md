# 排行榜勋章添加计划

## 用户选择：排名成就勋章

### 新增勋章列表

| 勋章ID | 名称 | 描述 | 稀有度 | 解锁条件 |
|--------|------|------|--------|----------|
| rank_top_10 | 榜上有名 | 进入排行榜前10名 | common | 任意排行榜前10 |
| rank_top_5 | 五强争霸 | 进入排行榜前5名 | rare | 任意排行榜前5 |
| rank_top_1 | 独占鳌头 | 登顶排行榜第一名 | epic | 任意排行榜第1 |
| rank_master | 榜首常客 | 同时在三个排行榜前10名 | epic | 三个榜单都在前10 |

## 实现步骤

### 步骤1：修改 badge.html
- 在 BADGES 数组添加4个新勋章定义（type: 'leaderboard'）
- 在 BADGE_ICONS 添加4个 SVG 图标
- 新增"🏆 排行榜成就"分类区块
- 修改 BadgeManager 添加排行榜判断逻辑

### 步骤2：修改 rank.html
- 添加保存用户排名数据到 localStorage 的功能
- 页面加载时保存当前用户在各榜单的排名

### 步骤3：修改 server.js
- 在排行榜 API 返回数据中包含当前用户的排名位置

### 步骤4：测试
- 刷新页面验证勋章显示
- 检查解锁逻辑是否正确
