# 添加争议话题按钮计划

## 目标
在主页右侧段位区域下方添加一个与六个功能按钮同样格式的"四大名著争议话题"按钮，并创建对应的页面。

## 实施步骤

### 1. 修改 home.html 布局
- 在 `.rank-section` 下方添加新的按钮区域
- 创建新的按钮样式（与现有按钮格式一致）
- 按钮链接到新的 `topic.html` 页面

### 2. 创建 topic.html 页面
- 创建基础 HTML 结构
- 添加返回主页按钮（右上角）
- 复制主页的 CSS 样式（背景、动画等）
- 预留内容区域（暂时填充占位内容）

### 3. 样式调整
- 确保新按钮与段位区域垂直对齐
- 调整间距，使布局协调
- 确保响应式设计正常工作

## 预期效果
```
┌──────────────┬──────────────┐
│  六个按钮    │   段位区域   │
│  （2x3）     │              │
│              │   争议话题   │
│              │    按钮      │
└──────────────┴──────────────┘
```
