# 自动显示解析功能实现计划

## 需求分析

实现"自动显示解析"按钮功能：
- **不按按钮**：所有题目作答后都不显示解析
- **按下按钮**：所有题目作答后显示解析（当前默认行为）

---

## 实现方案

### 1. 添加全局状态变量

在 JavaScript 变量区域添加：
```javascript
let autoShowAnalysis = false;
```

### 2. 修改按钮点击函数

修改 `openAutoTimer()` 函数：
- 切换 `autoShowAnalysis` 状态
- 更新按钮样式（选中/未选中）
- 不再显示 alert 提示

### 3. 修改设置页面按钮样式

让按钮能够显示选中状态（类似模式选择按钮的效果）

### 4. 修改答题逻辑

在以下函数中根据 `autoShowAnalysis` 决定是否显示解析：

#### 4.1 `selectOption()` - 选择题
- 只有 `autoShowAnalysis === true` 时才显示 `analysisBox`

#### 4.2 `submitFill()` - 填空题
- 只有 `autoShowAnalysis === true` 时才显示 `analysisBox`

#### 4.3 `submitBrief()` - 简答题
- 只有 `autoShowAnalysis === true` 时才显示 `analysisBox`

#### 4.4 `showQuestion()` - 恢复状态时
- 只有 `autoShowAnalysis === true` 且已作答时才显示 `analysisBox`
