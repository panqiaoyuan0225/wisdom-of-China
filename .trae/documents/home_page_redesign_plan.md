# 主页面重新设计 - 古典中国风方案

## 设计目标

将主页面改造为具有浓厚中国传统文化氛围的学习平台，提升视觉美感和用户体验。

## 设计主题

**"书香门第，文墨飘香"** - 古典中国风

***

## 实施步骤

### 1. 背景设计

* 创建宣纸纹理背景（使用 CSS 渐变模拟）

* 添加淡墨山水画装饰（SVG 或 CSS 实现）

* 添加飘落的竹叶/花瓣动态效果（可选）

* 配色方案：

  * 主色：米白/宣纸色 (#f5f5dc, #efe8d0)

  * 辅色：朱红 (#c41e3a)、金色 (#d4af37)、墨黑 (#2c2c2c)

  * 点缀色：竹青 (#7CB342)、靛蓝 (#5C6BC0)

### 2. 标题区域改造

* 添加书法风格标题

  * 使用 Google Fonts 或系统书法字体

* 添加副标题："品读经典，传承文化"

* 添加顶部装饰条（祥云/回纹图案）

* 添加欢迎用户信息（动态显示用户名）

### 4. 功能按钮重新设计

* 采用卡片式设计

  * 圆角矩形卡片

  * 渐变背景（与登录页风格一致）

  * 添加悬停阴影和上浮效果

* 优化按钮布局

  * 保持 3x2 网格布局

  * 增加按钮间距

  * 添加装饰性分隔线

* 统一视觉风格

  * 棕色 + 金色渐变

  * 白色文字

  * 精美图标

### 5. 菜单系统优化

* 保留右上角三点菜单

* 优化下拉菜单样式

  * 添加中国风边框

  * 优化动画过渡

### 6. 添加页脚

* 版权信息

* 简洁的装饰线

### 7. 响应式优化

* 确保移动端显示效果

* 优化小屏幕布局

* 保持动画性能

***

## 技术实现

### CSS 技术

* CSS Grid/Flexbox 布局

* CSS 渐变（linear-gradient, radial-gradient）

* CSS 动画（@keyframes）

* CSS 变量（方便主题管理）

* 媒体查询（响应式）

### SVG 技术

* 装饰性图案（祥云、回纹）

* 段位徽章（已存在，优化显示）

* 背景纹理

### JavaScript

* 用户信息加载

* 段位进度计算

* 动态效果控制

***
