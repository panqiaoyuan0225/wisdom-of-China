## 任务概述

**5个专题**：
- 壹·黛玉进府
- 贰·宝黛初见
- 叁·金玉良缘
- 肆·大观园诗社
- 伍·菊花诗会

**配色方案**：
| 元素 | 颜色 |

| 活跃状态背景 | 金色 `#d4af37` |
| 活跃状态文字 | 深粉色 `#8b1a4a` |

| 内容区背景 | 淡粉色 `rgba(255, 245, 245, 0.95)` |
| 标题颜色 | 绛红 `#b03060` |

## 实施步骤

### Step 1: 创建 honglou_stage1.html
1. 复制 xiyou_stage1.html 基础结构
2. 修改配色为粉色+金色主题
3. 设置标题为"大观寻梦"
4. 创建5个专题导航项
5. 编写各专题内容（黛玉进府、宝黛初见、金玉良缘、大观园诗社、菊花诗会）
6. 添加JavaScript切换功能

### Step 2: 创建 honglou_stage2.html
1. 使用相同结构框架
2. 设置标题为"烈火烹油"
3. 创建占位导航项
4. 内容区域显示"敬请期待"

### Step 3: 创建 honglou_stage3.html
1. 使用相同结构框架
2. 设置标题为"万艳同悲"
3. 创建占位导航项
4. 内容区域显示"敬请期待"
