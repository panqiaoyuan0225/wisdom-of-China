## 二、功能特性

1. **进度显示**：每个勋章显示当前进度（如：15/100 题）
2. **稀有度系统**：普通、稀有、史诗三个等级，不同颜色和特效
3. **通知提醒**：获得勋章时弹出动画通知
4. **自动标记已读**：用户浏览完专题页面后，系统自动标记为已读（无需手动点击按钮）

## 三、自动标记已读实现方案

### 方案：基于页面停留时间 + 滚动检测

当用户满足以下条件时，自动标记为已读：
1. 页面停留时间超过 30 秒
2. 页面滚动到底部（或滚动比例超过 80%）

### 实现逻辑

```javascript
// 在每个stage页面添加自动标记逻辑
let startTime = Date.now();
let hasScrolledToBottom = false;
const MIN_READ_TIME = 30000; // 30秒
const SCROLL_THRESHOLD = 0.8; // 滚动80%

// 检测滚动
window.addEventListener('scroll', function() {
    const scrollPercent = (window.scrollY + window.innerHeight) / document.body.scrollHeight;
    if (scrollPercent >= SCROLL_THRESHOLD) {
        hasScrolledToBottom = true;
        checkAndMarkAsRead();
    }
});

// 页面离开时检查
window.addEventListener('beforeunload', function() {
    const readTime = Date.now() - startTime;
    if (readTime >= MIN_READ_TIME && hasScrolledToBottom) {
        markStageRead(book, stage);
    }
});

// 页面可见性变化时检查
document.addEventListener('visibilitychange', function() {
    if (document.hidden) {
        const readTime = Date.now() - startTime;
        if (readTime >= MIN_READ_TIME && hasScrolledToBottom) {
            markStageRead(book, stage);
        }
    }
});
```
