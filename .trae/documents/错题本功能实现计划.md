# 错题本功能实现计划

## 一、功能需求分析

### 1.1 核心功能
- **记录错题**：用户答错的题目自动加入错题本
- **记录跳过题目**：用户跳过未作答的题目也加入错题本
- **持久化存储**：使用 localStorage 保存错题数据，刷新页面不丢失
- **查看错题本**：提供界面展示所有错题和跳过的题目
- **错题复习**：支持从错题本中选择题目进行复习

### 1.2 数据结构设计
```javascript
// 错题数据结构
{
    question: { /* 原题目对象 */ },
    mode: 'review' | 'extend',  // 题目来源模式
    type: 'wrong' | 'skipped',  // 错题类型：答错/跳过
    userAnswer: '',             // 用户答案
    timestamp: Date,            // 记录时间
    count: 1                    // 错误次数
}
```

## 二、实现步骤

### 步骤1：添加错题本数据管理模块
- 创建 `WrongBookManager` 对象
- 实现 `addWrongQuestion()` 添加错题
- 实现 `addSkippedQuestion()` 添加跳过题目
- 实现 `getWrongQuestions()` 获取所有错题
- 实现 `removeWrongQuestion()` 移除错题（答对后）
- 实现 `clearWrongBook()` 清空错题本

### 步骤2：修改答题逻辑
- 在 `selectOption()` 中，答错时调用添加错题
- 在 `submitFill()` 中，答错时调用添加错题
- 在 `skipQuestion()` 中，跳过时调用添加跳过题目
- 在答对时，如果该题在错题本中，则移除

### 步骤3：创建错题本UI界面
- 添加错题本弹窗/面板的HTML结构
- 添加错题本相关CSS样式
- 实现错题列表展示
- 支持按类型筛选（错题/跳过）
- 支持按模式筛选（复习/拓展）

### 步骤4：实现错题复习功能
- 从错题本开始答题
- 错题本答题时，答对可移除错题
- 显示错题统计信息
