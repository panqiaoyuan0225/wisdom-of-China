# 答题设置页面优化计划

## 修改需求
1. **页面更宽敞**：移除 max-width 限制，适合电脑屏幕
2. **删除限时按钮**
3. **添加辅助按钮**：错题本、自动限时解析
4. **题目数量改为滑块**：金色横轴，范围 1-100 题

## 实现步骤

### 步骤 1：调整页面宽度
- 移除 `.settings-card` 的 `max-width: 500px`
- 设置更宽的宽度，如 `max-width: 700px` 或 `800px`
- 增大内边距

### 步骤 2：删除限时按钮区域
- 删除限时按钮的 HTML 和 CSS
- 删除相关的 JavaScript 代码

### 步骤 3：实现金色滑块
- 使用 `<input type="range">` 实现滑块
- 自定义滑块样式：金色轨道、金色滑块
- 显示当前数值
- 范围 1-100

### 步骤 4：添加辅助按钮
- 错题本按钮：点击提示"错题本功能开发中"
- 自动限时解析按钮：点击提示"自动限时解析功能开发中"
- 样式与模式按钮一致

### 步骤 5：更新 JavaScript
- 删除 timer 相关代码
- 添加滑块值读取
- 添加辅助按钮点击事件

## 滑块样式设计

```css
/* 金色滑块轨道 */
input[type="range"] {
    -webkit-appearance: none;
    width: 100%;
    height: 8px;
    background: linear-gradient(to right, #d4af37, #f0d78c);
    border-radius: 5px;
}

/* 金色滑块按钮 */
input[type="range"]::-webkit-slider-thumb {
    -webkit-appearance: none;
    width: 24px;
    height: 24px;
    background: #d4af37;
    border-radius: 50%;
    cursor: pointer;
    box-shadow: 0 2px 6px rgba(0,0,0,0.2);
}
```
