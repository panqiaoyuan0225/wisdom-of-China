# 结果页面布局调整计划

## 需求分析

修改结果页面布局：
1. 将分数(如 2/5)和底部按钮放到页面正中间
2. 在分数左右各放一个礼炮emoji 🎉

---

## 实现方案

### 1. 修改HTML结构

将：
```html
<div class="result-container" id="resultContainer">
    <div class="result-score" id="resultScore">0/10</div>
    <div class="result-text" id="resultText">正确率：0%</div>
    <div>
        <button class="result-btn" onclick="restartQuiz()">重新答题</button>
        <button class="result-btn" onclick="backToSettings()">返回设置</button>
    </div>
</div>
```

改为：
```html
<div class="result-container" id="resultContainer">
    <div class="result-content">
        <span class="result-emoji">🎉</span>
        <div class="result-center">
            <div class="result-score" id="resultScore">0/10</div>
            <div class="result-text" id="resultText">正确率：0%</div>
            <div class="result-buttons">
                <button class="result-btn" onclick="restartQuiz()">重新答题</button>
                <button class="result-btn" onclick="backToSettings()">返回设置</button>
            </div>
        </div>
        <span class="result-emoji">🎉</span>
    </div>
</div>
```

### 2. 修改CSS样式

添加新的样式：
- `.result-content` - flex布局，水平居中
- `.result-emoji` - 礼炮emoji样式
- `.result-center` - 中间内容区域
- `.result-buttons` - 按钮容器
