# 段位系统实现计划

## 目标
为学习网站实现古典风格的段位系统，基于题目覆盖率升级，使用 SVG 设计独特的段位图标。

## 段位设计

| 段位 | 图标元素 | 色调 | 升级条件 |
|------|----------|------|----------|
| 蒙童 Ⅰ/Ⅱ/Ⅲ | 朱砂红点 + 竹简 | 青草绿 | 5% / 10% / 15% |
| 书生 Ⅰ/Ⅱ/Ⅲ | 青油灯 + 狼毫笔 | 素布蓝 | 20% / 28% / 36% |
| 秀才 Ⅰ/Ⅱ/Ⅲ | 折扇 + 儒巾 | 月白色 | 45% / 54% / 63% |
| 举人 Ⅰ/Ⅱ/Ⅲ | 红灯笼 + 马鞭 | 朱砂红 | 70% / 77% / 84% |
| 进士 Ⅰ/Ⅱ/Ⅲ | 宫花 + 雁塔 | 丁香紫 | 88% / 92% / 96% |
| 翰林 Ⅰ/Ⅱ/Ⅲ | 玉如意 + 墨宝 | 翡翠绿 | 97% / 98% / 99% |
| 状元 | 金殿冠冕 + 麒麟 | 流光金 | 100% |

## 图标设计风格
采用古典纹样风，使用 SVG 绘制：
- 简洁的线条勾勒主体
- 渐变填充增加层次感
- 细节纹理体现古典韵味

## 实现步骤

### Step 1: 设计 SVG 段位图标
为每个段位设计独特的 SVG 图标：
- 蒙童：圆形朱砂点 + 竹简纹理背景
- 书生：油灯火焰 + 笔架轮廓
- 秀才：半开折扇 + 扇骨细节
- 举人：灯笼主体 + 流苏装饰
- 进士：宫花造型 + 雁塔剪影
- 翰林：玉如意 + 宣纸纹理
- 状元：华丽冠冕 + 龙纹背景

### Step 2: 创建段位计算逻辑
- 从 localStorage 读取答题历史数据
- 计算已掌握题目数（正确率 ≥ 80%）
- 根据覆盖率确定当前段位

### Step 3: 实现段位页面 UI
- 顶部显示当前段位大卡片（图标 + 名称 + 进度）
- 中间显示升级进度条
- 底部显示所有段位列表（已解锁高亮，未解锁灰显）

### Step 4: 添加段位徽章到主页
- 在 index.html 显示当前段位小徽章
