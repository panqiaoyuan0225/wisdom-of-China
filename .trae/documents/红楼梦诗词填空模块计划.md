# 红楼梦诗词填空模块计划

## 目标
在 honglou_stage2.html 中新增"陆 · 诗词填空"模块，仿照 sanguo_stage3 的"必背古诗"功能。

## 实现内容

### 1. 新增导航项
在侧边栏添加：
```
<div class="nav-item" onclick="changeTab(this, '陆 · 诗词填空')">陆·诗词填空</div>
```

### 2. 新增CSS样式
```css
.poem-card { 诗词卡片样式 }
.poem-content { 诗词内容样式 }
.dictation-box { 默写框样式 }
.dictation-input { 输入框样式 }
.result-display { 结果显示样式 }
.correct { 正确文字样式 }
.wrong { 错误文字样式 }
.btn-group { 按钮组样式 }
```

### 3. 诗词内容（红楼梦经典诗词）

- **《葬花吟》** - 林黛玉
- **《秋窗风雨夕》** - 林黛玉
- **《咏白海棠》** - 林黛玉
- **《咏菊》** - 林黛玉
- **《题帕三绝》** - 林黛玉
- **《好了歌》** - 跛足道人
- **《飞鸟各投林》** - 结尾诗

### 4. JavaScript功能
- `poemLibrary` - 诗词库对象
- `openDict(id)` - 开启默写模式（隐藏原文）
- `closeDict(id)` - 返回阅读模式（恢复原文）
- `checkPoem(id)` - 批改功能（对比正确/错误）

### 5. 功能特点
- 点击"开启默写"后隐藏原文，防止偷看
- 支持标点符号自动跳过
- 正确文字显示绿色，错误文字显示红色下划线
- 缺漏文字显示○符号
